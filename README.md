# ornabot
 bot for orna through scrcpy

Commands:
battery - check battery
pause - pause/unpause bot
screen - screenshot
zoom - zoom out
coin - use coins and torch
exp - use exp pot
start - start orna
stop - close orna
lock - lock/unlock phone
c - connect scrcpy
dc - disconnect scrcpy
